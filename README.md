# Serverless Inference Inconsistency Demo

This Python script simulates a serverless machine learning inference scenario. It demonstrates how subtle variations in execution time and output can occur even with the same input, mimicking real-world serverless inference inconsistencies.

## Language

`python`

## How to Run

Save the code as `inference_demo.py` and run it from your terminal using `python inference_demo.py`.
